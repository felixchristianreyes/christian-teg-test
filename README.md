Step to run locally:

Step 1: 
npm install

Step 2: 
Create .env file with
VITE_EVENTS_LISTING_API_URL="https://teg-coding-challenge.s3.ap-southeast-2.amazonaws.com/events/event-data.json"

Step 3:
npm run dev

React, Vite, TailwindCSS, React Router, ShadCN, Typescript
